# Domain-Adaptation Evaluation

This directory contains the evaluation of domain adaptation by comparing PCA projections of embeddings from the pretrained and domain-adapted MacBERTh models. The approach follows [Qiu and Xu (2022)](https://www.researchgate.net/doi/10.13140/RG.2.2.14905.44649), adapted from their [diachronicBert PCA code](https://github.com/wendyqiu/diachronicBert/blob/main/postprocess/single_test/PCA.py).

This evaluation corresponds to the **Appendix (Domain-Adaptation Evaluation)** and **Figures 10–11** in the paper.

## Files

| File | Description |
|---|---|
| `pca_pretrained_vs_adapted.py` | Main script: loads embeddings from both the pretrained and domain-adapted MacBERTh, fits a joint PCA on the combined embeddings, and plots them with distinct markers (circles = pretrained, triangles = adapted). Produces both per-decade plots and a combined all-decades plot for each target word |
| `pca_plots/` | Output directory containing all generated PCA plots, organised by commodity |

## Usage

Requires embedding files (HDF5 format) from both the pretrained and domain-adapted models, generated by the embedding extraction step (see `embedding_extraction/`).

```bash
python pca_pretrained_vs_adapted.py \
    --pretrained_dir /path/to/embeddings_macberth_pretrained \
    --adapted_dir /path/to/embeddings_macberth_finetuned \
    --output_dir pca_plots \
    --decades 1840 1850 1860 1870 1880 1890 1900 1910 \
    --words coffee tea sugar opium cocoa tobacco \
    --sample_size 200
```

| Argument | Description | Default |
|---|---|---|
| `--pretrained_dir` | Directory containing HDF5 embedding files from pretrained MacBERTh | (required) |
| `--adapted_dir` | Directory containing HDF5 embedding files from domain-adapted MacBERTh | (required) |
| `--output_dir` | Where to save the plots | `pca_plots` |
| `--decades` | Decades to include | 1840–1910 |
| `--words` | Target commodity terms | coffee, tea, sugar, opium, cocoa, tobacco |
| `--sample_size` | Max usages to sample per word-decade pair (0 = use all) | 0 |

## Method

For each target word, the script:

1. Loads last-hidden-layer embeddings (768-dimensional) from both the pretrained and domain-adapted MacBERTh for all decades
2. Optionally subsamples to `--sample_size` usages per word-decade pair for readability
3. Fits a **joint PCA** on the combined pretrained + adapted embeddings, projecting to 2 dimensions
4. Plots pretrained points (circles) and adapted points (triangles), coloured by decade

The joint PCA means both models are projected into the same 2D space, making their distributional differences directly visible. As reported in the paper, the first principal component (42–49% of variance) typically separates the two models globally, while the adapted model shows greater internal substructure (multiple density peaks) within its region.

## Embedding file format

The script expects HDF5 files named `commodity_embeddings_{decade}.h5` with the structure:

```
commodity_embeddings_1840.h5
├── coffee/
│   ├── usage_0/
│   │   └── embedding    # shape: (768,)
│   ├── usage_1/
│   │   └── embedding
│   └── ...
├── tea/
│   └── ...
└── ...
```

## Relationship to the paper

- **Appendix (Domain-Adaptation Evaluation)**: the `_all_decades.png` plots for coffee and tea are presented as Figures 10 and 11, showing that the adapted model recovers more substructure than the pretrained model
- **Section 5** (Results): states that "all contextualised embedding analyses reported below use the domain-adapted MacBERTh," with this evaluation as the justification
- The per-decade plots (e.g., `coffee_1840s.png`) are not included in the paper but are available here for inspection
